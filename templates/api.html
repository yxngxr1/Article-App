Api Article App

Получение данных статей

  1. Список статей
    Для получения списка всех статей нужно совершить "get" запрос вида: /api/articles
    Ответ на запрос будет возвращен в Json формате со следующими полями обернутыми в "articles"
    # Поле Тип Значение
    1 id integer Уникальный идентификатор статьи, который указывается для обозначения и нахождения статей
    2 title string Название статьи
    3 description string Краткое описание статьи
    4 user_id integer Уникальный идентификатор пользователя, создателя статьи
  2. Данные одной статьи
    Для получения подробной информации одной статьи нужно совершить "get" запрос вида: /api/articles/{ id - статьи }
    Например: /api/articles/2
    Ответ на запрос будет возвращен в Json формате со следующими полями обернутыми в "article"
    # Поле Тип Значение
    1 id integer Уникальный идентификатор статьи
    2 title string Название статьи
    3 description string Краткое описание статьи
    4 content string Тело статьи в формате html разметки
    5 category_id integer Категория статьи. Соответсвие id и названий категорий смотреть ниже
    6 photo_url string Ссылка на картинку обложки статьи
    7 created_date string Дата создания статьи в формате: YYYY-MM-DD hh:mm
    8 is_private bool Значение, которое определяет, публичная статья или нет
    9 rating integer Рейтинг статьи
    10 views integer Количесвто просмотров статьи
    11 user_id integer Уникальный идентификатор пользователя, создателя статьи
    Соответсвие id и названий категорий
    id Название
    1 Наука
    2 Спорт
    3 Игры
    4 Программирование
    5 Фильмы
    6 Сериалы
    7 Аниме
    8 Обзоры
